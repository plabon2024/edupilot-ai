
# EduPilot AI

EduPilot AI is an AI-powered learning platform designed to help students study more effectively using document-based intelligence. Users can upload learning materials (such as PDFs) and automatically generate flashcards, quizzes, summaries, and interactive AI-driven explanations.

This repository is structured as a monorepo containing both the backend API and the frontend application.

## Key Features

* JWT-based authentication and user management
* PDF document upload and processing
* AI-powered learning tools:

  * Flashcard generation
  * Quiz generation and evaluation
  * Document summaries
  * Concept explanations
  * Context-aware AI chat
* Flashcard review and progress tracking
* Modular and scalable backend architecture
